## 📖 Overview

SmartMed AI is a modern healthcare web application built with **React.js** that helps users manage medications, monitor health metrics, and stay consistent with their wellness routine.

This project currently focuses on the **frontend UI/UX** with a clean, responsive design. Authentication is connected to the backend, while the remaining features are implemented as frontend interfaces.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/ashukr8287/SmartMedAI.git


Move into the project

```bash
cd SmartMedAI
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build project

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
