# 📌 About

Managing medicines regularly can be difficult, especially for elderly patients or people taking multiple medications every day.

**SmartMed AI** solves this problem by providing:

- 💊 Medicine Reminder Management
- ❤️ Health Monitoring Dashboard
- 🤖 AI-based Health Insights
- 🔐 Secure User Authentication
- 📱 Responsive User Interface

The project was built as a college project by **Ashu**, **Aman**, and **Anish** to demonstrate how modern web technologies can improve healthcare management.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/SmartMedAI.git

cd SmartMedAI
```

---

## Install Frontend

```bash
cd client

npm install
```

---

## Install Backend

```bash
cd server

npm install
```

---

## Run Frontend

```bash
npm run dev
```

---

## Run Backend

```bash
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

CLIENT_URL=http://localhost:5173
```
